# 📝 Full-Stack Task Management System

A simplified Trello clone built with the **MERN Stack**. This system allows users to organize their work through Boards, Lists, and Tasks with a modern dark UI.

---

## ✨ Features

* **User Auth:** Secure Login/Signup using **JWT** & **Bcrypt**.
* **Board Management:** Create multiple boards for different projects.
* **Organized Workflow:** Manage tasks inside lists (To Do, In Progress, Done).
* **Smart Editing:** Inline title editing and full Task management (CRUD).
* **Automated Emails:** Welcome email sent automatically upon registration.

---

## 🛠️ Tech Stack

* **Frontend:** React.js, CSS3, Axios.
* **Backend:** Node.js, Express.js.
* **Database:** MongoDB & Mongoose.
* **Auth:** JSON Web Tokens (JWT).

---

## 🚀 How to Run

### 1. Backend (Server)
1. Go to `cd server` and run `npm install`.
2. Create a `.env` file and add your (Mongo URI, JWT Secret, and Email credentials).
3. Start the server: `npm run dev`.

### 2. Frontend (Client)
1. Go to `cd Frontend` and run `npm install`.
2. Start the app: `npm start`.

---

## 📂 Project Structure
* `/Frontend`: React components, Pages, and Styling.
* `/Backend`: API Routes, Controllers, and Database Models.
